# 🍕 Pizza Ordering System

This is a simple Pizza Ordering System built with C# and Windows Forms.

## Features
- Select pizza size (Small, Medium, Large, Extra Large)
- Enter quantity
- Calculate total price
- Display orders in a table
- Print receipt 🧾
